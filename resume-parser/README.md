# TO RUN THE APPLICATION FOR FIRST TIME
## Installation in Frontend Folder

```bash
cd resume-parser
npm install 
npm start
```

## Installation in Backend Folder

```bash
py -3 -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```
